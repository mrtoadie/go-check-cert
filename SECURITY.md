# Security Policy

## Security & Privacy

- **Local deployment**: No external databases or cloud dependencies. All data remains local and is not transmitted.
- **Encrypted connection**: Built-in support for HTTPS with self-signed certificates or automatic setup via Let's Encrypt to secure the dashboard connection.
