# Contributions Welcome! 🚀

Thank you for your interest in helping to improve this project! Our goal is to create a fast, secure, and privacy-friendly dashboard for GitHub statistics. Whether you find bugs, suggest new features, or optimize the code—every contribution counts.

## How you can contribute

### 1. Open issues
Have you found a bug or have an idea for a new feature?
- First, check the [Issue Tracker](https://github.com/mrtoadie/go-check-cert/issues) to see if the issue has already been reported.
- Create a new issue with a clear description.
  - For **bugs**: Describe the bug, the steps to reproduce it, and your operating system.
  - For **features**: Explain the use case and why it helps the project.

### 2. Contributing Code
We prefer small, focused pull requests (PRs).

**The workflow:**
1. **Fork** the repository.
2. Create a **feature branch** from `main`:

   git checkout -b feature/your-feature-name
